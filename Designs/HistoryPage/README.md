关于 HistoryPage
=
Page 的 title 应该是<br>
>历史记录<br>
歷程記錄<br>
History><br>
<br>
History 里面的每一项历史记录轻触可以跳转，
长按如 Windows Phone 的应用界面一样，
可以固定磁贴到 “开始” 屏幕。<br>
HistoryPage 有 ApplicationBar，
上面应该有一个 iconButton，
icon 是一个 Windows Phone SDK 自带的垃圾桶 ApplicationBar icon，
下面标注的文字是<br>
清空历史记录<br>
清空歷程記錄<br>
Clear Histories<br>
<br>
点触后弹出一个 MsgBox，
title 是 2Jumpr，
内容是<br>
确认清除所有历史记录吗？<br>
確認清理所有歷程記錄嗎？<br>
Sure to clear all histories?<br>
<br>
OK 便清除所有的历史记录。
